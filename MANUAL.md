## 1. デモの流れ（概要）

```
操作担当者                          参加者
    |                                 |
    | アプリを起動してURLを共有 ──────→ QRまたはURLでアクセス
    |                                 |
    |                           カメラで顔を撮影
    |                                 |
    |                           バスケミニゲームで待機
    |                                 |
    |                           結果動画を視聴
    |                                 |
    |                           QRコードでスマホにDL
```

処理時間の目安: **撮影から結果まで 30秒〜1分**

## 3. 毎回の起動手順

### Colab の場合

**Step 1: ノートブックを開く**

Google Drive の `faceswap/` フォルダ内にある `colab_faceswap.ipynb` をダブルクリック
→ Google Colab で開く

**Step 2: GPU ランタイムを確認**

メニュー `Runtime` → `Change runtime type` → `T4 GPU` を選択して保存

> ⚠️ GPU を選ばないと処理が10倍以上遅くなります

**Step 3: NGROK_TOKEN を設定**

Cell 3 を開き、以下の行を書き換える:
```python
NGROK_TOKEN = 'ここにトークンを貼り付け'
```

**Step 4: 全セルを実行**

メニュー `Runtime` → `Run all`

最後のセル（Cell 6）に以下のような URL が表示されれば起動完了:
```
🎉 アプリURL: https://xxxx-xx-xx-xxx.ngrok-free.app
```

**Step 5: URL を参加者に共有**

表示された URL を:
- プロジェクターに表示してQRにする
- LINE/チャットで送る
- 口頭で伝える

などの方法で参加者に共有してください。

> 💡 `/?mode=stg` を URL の末尾につけると、カメラなしで画像ファイルをアップロードできるテストモードになります。

---

### Windows PC の場合

**Step 1: `start.bat` をダブルクリック**

初回は自動で環境セットアップとモデルダウンロードが実行されます（数分かかります）。

**Step 2: URL を確認**

コンソールに以下のような行が表示されれば起動完了:
```
https://xxxx.trycloudflare.com
```

**Step 3: URL を参加者に共有**（Colab と同じ）

## 4. 参加者の操作手順

> **参加者に見せる用の手順です。印刷して渡すか、口頭でガイドしてください。**

---

### STEP 1: アクセス

スマホまたは PC のブラウザで共有された URL を開く。

---

### STEP 2: START

画面中央の **「START」** ボタンをタップ。

---

### STEP 3: 顔を撮影

カメラが起動したら、**顔が画面中央に来るよう**に調整して **「📷 撮影」** をタップ。

> - 明るい場所で撮影すると精度が上がります
> - 正面を向いて撮影してください
> - 眼鏡・マスクがあると認識しにくい場合があります

撮影後、確認画面が表示されます。
- 顔がはっきり写っていれば **「これでOK」**
- 撮り直したい場合は **「撮り直す」**

---

### STEP 4: ミニゲームで待機

処理中にバスケのミニゲームが始まります。

カメラに向かって:
- 🏀 **ドリブル**: 手首を上下に動かす
- 🙌 **シュート**: 両手を頭より上に上げる

画面下のプログレスバーが 100% になると結果が表示されます。

---

### STEP 5: 結果を確認

合成された動画が自動再生されます。

**スマホにダウンロードするには:**
画面の QR コードをスマホのカメラで読み取る → 動画がダウンロードされます。

---

### STEP 6: もう一回

**「もう一回」** ボタンで最初からやり直せます。

---

## 5. トラブルシューティング

### 顔が検出されないエラーが出る

- 撮影場所が暗い → 明るい場所で撮り直し
- 顔が小さすぎる / 横を向いている → 正面から大きく映るよう撮り直し
- 「撮り直す」ボタンで再撮影

### 処理が止まっている・長すぎる

- Colab の場合: Cell 5 のサーバーログを確認（セルの出力を見る）
- 5分以上経っても変化がない場合は Cell 5 を再実行

### カメラが起動しない

- ブラウザがカメラのアクセスを求めている → 「許可」をタップ
- スマホの場合はブラウザアプリのカメラ許可を設定から確認
- URL の末尾に `?mode=stg` を追加すれば画像ファイルのアップロードで代用可能

### QR コードを読み取れない

- QR が小さい場合はスマホを画面に近づける
- スマホのカメラアプリ（またはQRリーダー）で読み取る
- URL を直接コピーして送る方法でも可

### Colab のセッションが切れた

Colab は一定時間でセッションが切断されます（無料版は90分アイドルで切断）。

復旧方法:
1. `Runtime` → `Run all` で全セル再実行
2. モデルは Drive に保存済みなので再ダウンロードは不要（1〜2分で復旧）

### ngrok の URL が変わった

起動するたびに URL が変わります。Cell 6 に表示された最新の URL を参加者に共有し直してください。
